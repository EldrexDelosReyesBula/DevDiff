# WCAG 2.2 AA Compliance

DevDiff targets WCAG 2.2 AA level accessibility compliance.

## Standards Followed

- **High Contrast**: Sleek, high-contrast dark modes for visual excellence without straining eyes.
- **Reduced Motion**: Respects `prefers-reduced-motion` system preferences.
- **Semantic HTML**: Semantic structures ensure screen reader parsing works out of the box.
