# Privacy Guarantees

We take your privacy seriously. Here is what we guarantee:

- **No Telemetry:** DevDiff does not phone home, report usage logs, track CLI actions, or collect repository statistics.
- **BYOAI:** You choose the AI. If using Ollama or other local models, your code analysis is 100% private to your machine.
- **Transparent Prompting:** You can inspect exactly what is sent to the models by running in dry-run mode (`devdiff generate --dry-run`).
