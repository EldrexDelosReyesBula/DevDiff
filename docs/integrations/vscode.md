# VS Code Extension

The DevDiff VS Code extension (`@eldrex/vscode`) brings AI-powered changelog explanations directly to your editor.

## Key Features

* **Status Bar Button:** One-click **DevDiff** button in the VS Code status bar to instantly explain staged Git changes.
* **Sidebar Webview:** Interactive dashboard showing active files, change statistics, and AI-generated explanations.
* **Inline Diffs:** AI explanations integrated directly into file diff views.
* **Quick Explain:** Explain staged changes without leaving VS Code.
* **Marketplace Integration:** Install directly from the VS Code Marketplace.

Install the extension from the VS Code Marketplace:
